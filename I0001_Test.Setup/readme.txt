VS2010 中.创建安装包的步骤:


1.解决方案资源管理器中，选择已有的解决方案，鼠标右键。 选择 添加-->新建项目


2.弹出对话框中
  左边选择: 其他项目类型 --> 安装和部署 --> Visual Studio Installer 
  右边选择：安装项目.


3.在安装项目的 文件系统视图里面，选择 “应用程序文件夹”， 鼠标右键。 选择 添加 --> 项目输出。
  确认对话框中，使用默认选择的 “主输出”，  按“确定”按钮。


4.选择 “应用程序文件夹”， 选择项目的 主输出来自 ...  鼠标右键，选择 创建快捷方式。


5.选择创建出来的快捷方式，拖动到 用户的“程序”菜单
  然后通过鼠标左键单击的这个快捷方式， 然后修改名称.
  然后选择这个快捷方式，在属性窗口中，选择 Icon --> 浏览. 
  查找范围为 应用程序文件夹， 文件类型为 可执行文件 *.exe
  选择那个有图标的 exe 文件，就可以了。


6.解决方案资源管理器中，选择 安装项目， 鼠标右键，选择 属性
  点击 “系统必备” 按钮
  选择 “从与我的应用程序相同的位置下载系统必备组件”
  按“确定”按钮。

  
7.解决方案资源管理器中，选择 安装项目， 鼠标右键，选择 生成。
